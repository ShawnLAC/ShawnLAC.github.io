Amazon

介绍

  • 亚马逊是一家总部位于美国西雅图的跨国电子商务企业，目前是全球最大的互联网线上零售商之一。
  • Oncall

部门

湾区-AWS dp

  • WLB 很差 55h/week

AWS dp

  • 早点了解公司内的组，拿transfer
  • 推荐的AWS的组：S3, EC2, DDB

总结：

  • 选组非常重要！
  • 亚麻不是特别适合new grad

Bloomberg

介绍

  • 成立于1981年的美国彭博资讯公司是目前全球最大的财经资讯公司。彭博仅用了22年的时间，就将它的金融数据市场的销售收入超越了具有150年历史的、世界上最大的资讯公司——路透集团。
