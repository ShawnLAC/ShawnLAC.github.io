START MY USC STUDY JOURNEY

发表于 2021-08-19

USC I’m coming!

大连-首尔-洛杉矶

这一天是2021年8月6日，美国新增人数为：168351。这个数字背后，承载了一个个留学生家庭心中的忐忑，但同时，也正是这样的大环境，让我的这次留学之旅显得更加珍贵，想想古代的历朝历代，战争和草菅人命的世道往往是主旋律，想想近代的救亡图存，离不开人民在枪林弹雨中奔涌的热忱。嗯，时代变了，但也没什么不一样的。以积极的心态面对时代抛出的难题，才有可能迎刃而解。

尼采说：“Those who were seen dancing were thought to insane by those who could not hear the music.” 没有事实，只有诠释。不忘初心，有所收获。

入境插曲：被关小黑屋2.5小时，问题不大，但是考验心态 :(

PPV

受到新冠疫情的影响，今年的PPV可以线上进行。
这个需要在入境美国后尽快完成，以消除会限制选课的STU50的hold。

TB-test

TB检测必须在美国境内进行，我是在Lyon Center抽血检测的，推荐预约在清晨，排队的人少。

Covid-19检测

落地LA之后3天内需要检测一次，而后是7天一次。

公寓

入住的具体位置是，来了之后需要去洛杉矶能源中心注册一下电费的账户。对公寓还是很满意的，很干净整洁，位置也比较安全。

选课

选课是要狠狠恨斗两句的，USC的人实在太多了，很多课申请DC的人数都排出去几间教室，本着不想水的想法，希望最后不要只剩下预留的CSCI585了。。

shopping

目前比较推荐的：

  • 杂货、果汁、蔬菜：Raphs(in-person)
  • 肉、菜、蛋、酒：Trader Joes(in-person)
  • 日用品：Target(shipping or in-person)
  • 电子产品、日用品：Amazon(shipping 但是送货时间很不规律)
  • 食材：weee!(shipping 平均两天送达)
  • 华人商品：99 Ranches Market（in-person)

新冠

美国人的防疫意识很差，自己做好防护吧！
贴几个有用的网站：

  • 学校记录新增人数的：https://coronavirus.usc.edu/dashboard/

Hexo安装小白教程

  • link：
    https://www.bilibili.com/video/BV1Yb411a7ty?from=search&seid=10067804130876709922

感谢CodeSheep!

准备工作

node -v 查看node版本
npm -v 查看npm版本
npm install -g cnpm –registry=http://registry.npm.taobao.org 安装淘宝的cnpm 管理器
cnpm -v 查看cnpm版本
cnpm install -g hexo-cli 安装hexo框架
hexo -v 查看hexo版本

初始化

mkdir blog 创建blog目录
cd blog 进入blog目录
sudo hexo init 生成博客 初始化博客
hexo s 启动本地博客服务
http://localhost:4000/ 本地访问地址
hexo n 我的第一篇文章 创建新的文章
返回blog目录
hexo clean 清理
hexo g 生成
Github创建一个新的仓库 YourGithubName.github.io
cnpm install –save hexo-deployer-git 在blog目录下安装git部署插件

配置_config.yml

# Deployment
## Docs: https://hexo.io/docs/deployment.html
deploy:
      type: git
     repo: https://github.com/YourGithubName/YourGithubName.github.io.git
      branch: master

hexo d 部署到Github仓库里

git clone https://github.com/litten/hexo-theme-yilia.git themes/next

修改hexo根目录下的 _config.yml 文件 ： theme: next

hexo c 清理缓存
hexo g 生成
hexo d 部署到远程Github仓库
